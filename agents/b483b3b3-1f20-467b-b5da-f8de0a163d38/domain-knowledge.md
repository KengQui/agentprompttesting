A) Purpose and Scope

This agent supports US benefits life change events (QLEs) for employees using UKG Ready ESS.

It explains QLE concepts, typical allowed changes, documentation expectations, and next steps.

Customer policy is the source of truth. If a rule isn’t available in knowledge/policy table, the agent must avoid guessing and escalate.

B) Core Definitions (what the agent “knows”)

Qualifying Life Event (QLE): a change in status that may allow benefits changes outside open enrollment.

Event date: when the life event happened (marriage date, birth date, coverage loss date).

Submission window: number of days after the event date in which changes must be submitted (commonly 30; sometimes 60; customer-specific).

Effective date: when coverage changes start; depends on plan/employer rules and event type.

Consistency rule: changes must be consistent with the event (e.g., add spouse after marriage, remove spouse after divorce).

C) Common QLE Types (US)

Marriage

Divorce/legal separation/annulment

Birth

Adoption/placement/foster placement

Death of spouse/dependent

Loss of other coverage / gain of other coverage

Move affecting plan availability / service area

Work status change affecting eligibility

Dependent eligibility change (age-out, etc.)

D) Typical Documentation (customer policy may differ)

Marriage: marriage certificate

Divorce: divorce decree/court order

Birth: hospital record or birth certificate

Adoption/placement: placement/adoption paperwork

Loss of coverage: termination letter, COBRA notice, letter of ineligibility

Move: proof of new address (if required)

Death: death certificate (if required)
